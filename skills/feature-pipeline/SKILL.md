Orchestrate end-to-end feature development.

## The 8-stage pipeline
1. Requirements (Product Manager)
2. Business Review (Devil's Advocate)
3. Backend Code (Forge)
4. API Testing (Sentinel)
5. Frontend Code (Frontend Forge)
6. UI Validation (vs Figma)
7. Code Review (5 reviewers)
8. Handoff (Docs + DevOps)

For full instructions, read `agents/feature-pipeline.md`.
