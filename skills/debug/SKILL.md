Analyze errors, stack traces, and bugs across any stack.

## How to use
Paste an error message, stack trace, or describe the bug. The agent will:
1. Read the file where the error occurs
2. Search codebase for related code
3. Diagnose the root cause
4. Show the exact fix (before → after)

Works with: .NET, NestJS, Django, Spring Boot, React, React Native, Flutter.

For full instructions, read `agents/debug.md`.
