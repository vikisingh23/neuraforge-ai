Create videos using Remotion, FFmpeg, and short-form generators.

## Tools available
- Remotion: React-based programmatic video (custom, data-driven)
- FFmpeg: trim, merge, transcode, subtitles, watermarks
- Media Processor: resize, thumbnails, metadata
- Short Video Maker: TikTok/Reels/Shorts with captions

For full instructions, read `agents/video-creator.md`.
