Migrate code between frameworks, languages, and patterns.

## Supported migrations
- JavaScript → TypeScript
- Class components → Hooks
- Express → NestJS
- REST → GraphQL
- setState → React Query / Riverpod
- Framework upgrades (NestJS 9→10, React 18→19, .NET 7→8)
