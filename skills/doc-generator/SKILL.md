Create PowerPoint, Word, and Excel documents.

## What it creates
- PPTX: presentations with slides, tables, images
- DOCX: documents with headings, paragraphs, tables, images
- XLSX: spreadsheets with multiple sheets, formulas, formatting
- CSV: data export/import

Uses @neuraforge/office-mcp (32 tools).

For full instructions, read `agents/doc-generator.md`.
